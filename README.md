# Otel Rezervasyon Sistemi

Bu proje, Algoritma ve Programlamaya Giriş dersi kapsamında geliştirilen
konsol tabanlı bir otel rezervasyon otomasyonudur.

Programda rezervasyon oluşturma, iptal etme, güncelleme, arama ve
listeleme işlemleri yapılabilmektedir. Menü yapısı kullanılmıştır
ve kullanıcı çıkış yapmadan program sonlanmaz.

Java dili kullanılarak geliştirilmiştir.
